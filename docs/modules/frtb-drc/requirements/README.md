# frtb-drc requirement registry

The executable DRC requirement registry is package-local:

- [`packages/frtb-drc/docs/requirements/BASEL_FRTB_DRC.yml`](../../../../packages/frtb-drc/docs/requirements/BASEL_FRTB_DRC.yml)

That file is the authoritative machine-readable requirement inventory for DRC.
It is code-coupled evidence and should not be duplicated under
`docs/modules/frtb-drc/requirements/`.
