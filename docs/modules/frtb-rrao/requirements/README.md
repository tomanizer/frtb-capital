# frtb-rrao requirement registry

The executable RRAO requirement registry is package-local:

- [`packages/frtb-rrao/docs/requirements/BASEL_FRTB_RRAO.yml`](../../../../packages/frtb-rrao/docs/requirements/BASEL_FRTB_RRAO.yml)

That file is the authoritative machine-readable requirement inventory for RRAO.
It is code-coupled evidence and should not be duplicated under
`docs/modules/frtb-rrao/requirements/`.
