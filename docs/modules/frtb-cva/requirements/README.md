# frtb-cva requirement registry

- [`packages/frtb-cva/docs/requirements/BASEL_FRTB_CVA.yml`](../../../../packages/frtb-cva/docs/requirements/BASEL_FRTB_CVA.yml)

`docs/modules/frtb-cva/requirements/`.
