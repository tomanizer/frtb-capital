# frtb-sbm requirement registry

The executable SBM requirement registry is package-local:

- [`packages/frtb-sbm/docs/requirements/BASEL_FRTB_SBM.yml`](../../../../packages/frtb-sbm/docs/requirements/BASEL_FRTB_SBM.yml)

That file is the authoritative machine-readable requirement inventory for SBM.
It is code-coupled evidence and should not be duplicated under
`docs/modules/frtb-sbm/requirements/`.
